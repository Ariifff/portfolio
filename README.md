## ✨ Features & Highlights

- **Interactive 3D CSS Android Phone Mockup:** A fully CSS-rendered 3D phone model in the Hero section and Project showcase cards. It features 3D perspective transforms, smooth custom animations, and a floating motion effect using **Framer Motion**.
- **Modern Premium Design:** Built with a glassmorphic aesthetic, sleek gradients, Outfit and Roboto Mono typography, and carefully curated Tailwind-adjacent styling values.
- **Micro-Animations & Transitions:** Fluid page scroll animations using **GSAP** and element-wise view animations using **Framer Motion**.
- **Smooth Custom Cursor:** A dual-ring cursor that tracks mouse movement with lag-compensation on desktop and seamlessly disables on touch screens for touch-friendly UX.
- **Fully Responsive Layout:** Optimized from the ground up for desktop, tablet, and mobile screens. Mockup models dynamically scale down using custom CSS properties to fit any viewport perfectly without overflowing.

- **Framework:** React 19 (Vite-based HMR)

- **Framer Motion:** Staggered transitions, hover scales, and mobile menu entry overlays.

## 🚀 Getting Started & Local Setup

To run this project locally, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed (v18 or higher recommended).

### 1. Clone the Repository
```bash
git clone https://github.com/Ariifff/portfolio.git
cd portfolio
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run Development Server
Start the local Vite dev server:
```bash
npm run dev
```
Open your browser and navigate to `http://localhost:5173/` to view the website.

### 4. Create Production Build
To build the project for hosting:
```bash
npm run build
```
This compiles the code and outputs optimized production-ready static assets into the `dist/` directory.
